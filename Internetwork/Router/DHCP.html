Dynamic Host Configuration Protocol

How to make a DHCP pool on a router
In this example, our computer will be connected to GigabitEthernet0/0

Router>enable
Router#conf term
Router(config)#interface GigabitEthernet0/0
Router(config-if)#no shutdown

GigabitEthernet0/0 will now be online but our computer cannot communicate with the router just yet as it needs an ip address, we will give the computer an ip address via DHCP.

Router(config-if)#ip dhcp excluded-address 192.168.0.1
Router(config-if)#ip dhcp pool routerPool
Router(dhcp-config)#network 192.168.0.0 255.255.255.0
Router(dhcp-config)#dns-server 8.8.8.8
Router(dhcp-config)#default-router 192.168.0.1
Router(dhcp-config)#exit

Our computer should now be able to get an ip address via DHCP
